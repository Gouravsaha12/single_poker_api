### Poker API
This repository contains a simple Poker API built with Node.js and Express. The API allows users to play poker with bots, manage game states, and retrieve game information.

### Features
- Start a new poker game with bots
- Retrieve the current state of the game
- Make moves (bet, fold, check, etc.)

### Technologies Used
- Node.js
- Express
- JavaScript
- RESTful API design
- bcrypt for password hashing
- JSON Web Tokens (JWT) for authentication
- MongoDB for data storage
- Jest for testing
